# **App Name**: CallSense AI

## Core Features:

- Audio Upload & Processing: Upload call recordings (.mp3, .wav) for analysis.
- Speech-to-Text Transcription: Transcribe audio files to text using Whisper.
- Sentiment Analysis: Analyze call transcripts to determine sentiment (positive, neutral, negative).
- Compliance Checker Tool: Detect compliance violations by matching transcribed text against a predefined list of keywords and phrases. Use the LLM as a tool.
- Agent Performance Scoring: Score agent performance based on sentiment and compliance metrics.
- Dashboard Display: Display analysis results on a dashboard with key metrics, charts, and reports.
- Reporting and Export: Export reports and analysis data in PDF and CSV formats.

## Style Guidelines:

- Primary color: Deep sky blue (#42A5F5). Sky blue suggests clarity and trustworthiness, which is vital for a QA application.
- Background color: Light blue (#E3F2FD). It's desaturated and bright, providing a neutral backdrop for the primary color and other elements.
- Accent color: Green-Blue (#29B6F6), a brighter color, placed 30 degrees to the 'left' of the primary color. It can bring more attention to CTAs.
- Body and headline font: 'Inter', a grotesque sans-serif font, should be used for a modern, neutral look that is suitable for both headlines and body text.
- Use simple, professional icons to represent different metrics and functions.
- Use a clean and organized layout with clear sections for each analysis component.
- Incorporate subtle animations for data updates and user interactions to improve engagement.
